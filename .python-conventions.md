# Python conventions — pdomain-book-tools

Read by the `writing-python` skill on invocation. Project deltas only; the procedure, the
strict baseline, and the gate-resolution order live in the plugin.

## Gate

- Run `make lint`, `make format`, and `make typecheck`; use `make test` only for
  the integration suite.
- Scoped tests: `uv run pytest tests/<area you changed> -q` (or `-k`). Full suite is
  integration-only, run SERIALLY.
